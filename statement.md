# Project Title:
Schrödinger Equation Visualizer for a Particle in a One-Dimensional Box

# Objective:
To computationally solve and visualize the wave functions and probability densities of a particle confined in a one-dimensional box using Python.

# Description:
In quantum mechanics, a particle confined in a 1D box has discrete energy levels and corresponding wave functions. This project computes and visualizes:

- Wave function ψ(x)
- Probability density ψ²(x)

for different quantum numbers (n).

# Inputs:
- Length of the box (L)
- Quantum number (n)

# Outputs:
- Graph of ψ(x) vs x
- Graph of ψ²(x) vs x

# Tools Used:
- Python
- NumPy
- Matplotlib

# Applications:
- Understanding quantum confinement
- Visualizing abstract quantum concepts
